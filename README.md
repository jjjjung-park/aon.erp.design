# aon.erp.design
피그마 플러그인
1. varToComponent
 - 변수의 갯수에 따라 원하는 인스턴스 반복 생성 및 원하는 string 프로퍼티와 연결
 - 테이블, 리스트와 같이 반복 작업이 많은 화면용

2. importVarSheet
 -  구글 시트에 정리 된 텍스트 긁어오는 플러그인
 -  규칙에 맞게 입력된 텍스트를 불러오기 후 변수 생성
 -  제일 첫번째 행을 기준으로 긁어옴
 -  name : 변수명 / type : 변수타입 / label : 변수 모드1 / value : 변수 모드2 (소문자)
 -  name은 비어있으면 label 내용으로 자동 번역 후 생성
 -  기획서<->디자인의 텍스트 간극 줄이는 플러그인
 -  싱크 맞추는 방법은 고민해야 함.

3. LayerTextToVariables
 - 기획서 내 테이블 구조 기준으로 텍스트 추출하는 플러그인
 - 구글 시트로 칼럼명 관리하기 전까지 사용될 텍스트 추출 임시방편
 - 기획서 내 테이블 레이어(컴포넌트, 프레임)선택 후 추출 클릭
 - 컬렉션 명 입력 가능 (기본 컬렉션명 있음)
 - prefix 입력으로 컬렉션 내에서 그룹 설정 가능
 - 추출 시 변수명은 자동 번역 됨
 - 번역 결과 수정 가능

4. SheetToComponent
- 구글 시트를 바탕으로 컴포넌트 생성
- 구글 시트 내에서 키워드 검색 가능
- 탭을 선택하면 해당 탭 안에서만 입력한 키워드로 검색
- 키워드 입력 없으면 선택한 탭의 모든 내용 불러옴
- 시트 내용 호출 후 선택한 인스턴스에 label, value 프로퍼티 검색
- 인스턴스 내 프로퍼티가 label이면 시트의 label 내용을 연결 (value면 value 연결)
- 플러그인 UI에서 체크된 키워드 갯수만큼 반복하여 인스턴스 생성
