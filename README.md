# aon.erp.design
피그마 플러그인
1. varToComponent
 - 변수의 갯수에 따라 원하는 인스턴스 반복 생성 및 원하는 string 프로퍼티와 연결
 - 테이블, 리스트와 같이 반복 작업이 많은 화면용

2. importVarSheet
 -  구글 시트에 정리 된 텍스트 긁어오는 플러그인
 -  규칙에 맞게 입력된 텍스트를 불러오기 후 변수 생성
 -  제일 첫번째 행을 기준으로 긁어옴
 -  name : 변수명 / type : 변수타입 / label : 변수 모드1 / value : 변수 모드2 (소문자)
 -  name은 비어있으면 label 내용으로 자동 번역 후 생성
 -  기획서<->디자인의 텍스트 간극 줄이는 플러그인
 -  싱크 맞추는 방법은 고민해야 함.
