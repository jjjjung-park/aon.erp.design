# Changelog

---

## [0.0.1] - 2026-05-15

### Added
- **다중 키워드 검색** — 쉼표(`,`)로 구분해 여러 키워드를 OR 조건으로 동시 검색
- **키워드 하이라이트** — 검색 결과에서 일치하는 키워드를 파란색으로 표시
- **선택 항목 태그** — 체크한 항목을 검색 결과 하단에 pill 태그로 표시, ✕ 클릭 시 선택 해제
- **신규 label 감지** — 스냅샷에 없는 새 행을 `[신규]` 뱃지로 표시
- **신규 선택 버튼** — 신규 행만 일괄 선택
- **삭제 감지 (취소선 방식)** — 시트에서 취소선 처리된 행을 `[삭제됨]` 뱃지로 표시, 체크박스 비활성화
- **하단 버튼 조건 분기** — 변경건만 선택 시 동기화 버튼, 그 외 생성 버튼으로 자동 전환
- **비활성 버튼 툴팁** — 행 선택 후 Figma 인스턴스 미선택 시 안내 문구 상시 표시

### Changed
- **키워드 검색 최소 글자 수** — 2글자 미만 키워드는 검색에서 제외 (오매칭 방지)
- **동기화 버튼 디자인** — 세컨더리 스타일(흰 배경 + 테두리)로 변경
- **섹션 타이틀 스타일** — 필수 섹션(1, 3)에 `*` 표시, 타이틀과 서브 라벨 시각 계층 구분
- **시트 데이터 파싱** — `spreadsheets.get + includeGridData` 방식으로 전환, 취소선 서식 인식 지원
- **한글 NFC 정규화** — 데이터 입수 시점에 `.normalize('NFC')` 적용으로 한글 키워드 오매칭 수정

### Fixed
- `insertChild` 오류 — 인스턴스 내부 노드 선택 시 편집 가능한 상위 프레임으로 자동 탈출 (`getCloneContext`)
- 페이지 직속 인스턴스 생성 방향 — 감싸는 프레임 없을 때 무조건 아래 방향으로 고정
- n번째 동기화 실패 — 스냅샷 업데이트 시 다른 시트 스냅샷을 덮어쓰던 버그 수정
- 특정 페이지 라벨 변경 감지 안 됨 — 깨진 인스턴스의 `componentProperties` 접근 오류를 `safeComponentProperties()`로 처리
- 한글 IME 마지막 키워드 미검색 — `@keyup.enter` 전환 및 `search()` 시작 시 `blur()` 호출로 조합 완료 보장
- 취소선 신규 행 `[신규]` 오표시 — 취소선 행은 신규 감지에서 제외
- 물리적 행 삭제 감지 제거 — 행 순서 변경 시 오탐 유발, 취소선 방식으로 통일
