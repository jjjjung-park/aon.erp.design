# SheetToComponent

구글 시트의 데이터를 읽어 피그마 컴포넌트 인스턴스에 자동으로 연결·생성하고, 시트 변경 시 동기화할 수 있는 피그마 플러그인입니다.

---

## 사전 준비

### 1. 구글 시트 형식

시트의 **첫 번째 행**은 반드시 아래 헤더를 포함해야 합니다. (대소문자 무관)

| name | type | label | value | description |
|------|------|-------|-------|-------------|
| 컴포넌트 식별자(선택) | STRING(고정) | 피그마에 표시할 라벨 텍스트 | 피그마에 표시할 값 텍스트 | 설명 (선택) |

**예시**

| name | type | label | value | description |
|------|------|-------|-------|-------------|
| kpi_01 | STRING | 매출 | ₩4.2억 | 월간 매출 |
| kpi_02 | STRING | 비용 | ₩1.8억 | 월간 비용 |
| kpi_03 | STRING | 전환율 | 3.2% | |

### 2. 피그마 컴포넌트 조건

플러그인은 인스턴스의 **컴포넌트 프로퍼티** 중 `label`, `value`, `description`으로 이름 붙은 TEXT 속성을 찾아 값을 주입합니다.

- 프로퍼티 이름은 대소문자 무관 (`Label`, `VALUE` 등 동일 처리)
- `label`과 `value`가 같은 인스턴스에 있어도 되고, 분리된 구조여도 동작

### 3. Google API Key

Google Sheets API v4를 사용합니다. [Google Cloud Console](https://console.cloud.google.com/)에서 API 키를 발급한 뒤, 해당 키에 **Google Sheets API** 권한을 부여하세요.

## 사용 방법

### STEP 1 — 시트 호출

1. 플러그인 실행 후 **구글 시트 URL**을 입력합니다.
   - 이전에 사용한 시트는 드롭다운에서 선택 가능
2. **Google API Key**를 입력합니다.
   - 한 번 저장된 키는 다음부터 자동 입력됨
   - 키를 바꾸려면 **API 키 변경** 체크박스를 체크
3. **시트 호출** 버튼을 클릭합니다.
   - 전체 탭의 모든 행을 한 번에 불러옵니다.
4. (선택) **탭 범위**에서 특정 탭만 선택하면 해당 탭 데이터만 표시됩니다.

---

### STEP 2 — 키워드 검색 (선택)

- 검색창에 텍스트를 입력하고 **검색** 버튼을 클릭합니다.
- `label`, `value` 텍스트를 기준으로 필터링됩니다.
- 키워드를 지우고 검색하면 전체 결과로 돌아옵니다.

---

### STEP 3 — 행 선택

불러온 결과 목록에서 피그마에 연결할 행을 체크합니다.

| 버튼 | 동작 |
|------|------|
| 수정건 선택 | label이 변경된 행만 자동 선택 (변경건이 있을 때 표시) |
| 전체선택 | 전체 행 선택 |
| 선택해제 | 선택 초기화 |

> **label 변경** 뱃지가 표시된 행은 이전 연결 시점 이후 label 텍스트가 바뀐 항목입니다.

---

### STEP 4 — 연결 대상 선택

피그마 캔버스에서 인스턴스(또는 인스턴스를 포함한 프레임)를 선택합니다.
선택된 레이어 정보가 **3) 연결 대상** 영역에 표시됩니다.

---

### STEP 5 — 생성 및 연결

**N개 생성 및 연결** 버튼을 클릭합니다.

- 선택한 레이어를 템플릿으로 삼아 행 수만큼 복제본을 생성합니다.
- 복제본의 생성 방향은 선택한 레이어의 부모 레이어에 설정된 오토 레이아웃 방향대로 생성됩니다.
- 각 복제본에 `label`, `value`, `description` 값이 자동으로 주입됩니다.
- 생성 시점의 시트 데이터가 **스냅샷**으로 저장되어 이후 변경 감지에 활용됩니다.

---

### STEP 6 — 시트 변경 감지 및 동기화

시트에서 label 값을 수정한 뒤 플러그인에서 시트를 다시 호출하면:

1. 변경된 행에 **label 변경** 뱃지가 표시됩니다.
2. 상태 메시지에 `⚠️ label 변경 N건 감지`가 표시됩니다.
3. **수정건 선택** 버튼으로 변경된 행만 한 번에 선택할 수 있습니다.
4. 풋터의 **⚡ N개 동기화** 버튼이 활성화됩니다.
5. 버튼을 클릭하면 피그마 캔버스의 해당 인스턴스 label 값이 자동으로 업데이트됩니다.

> 동기화 버튼은 label이 변경된 행을 1개 이상 선택했을 때만 활성화됩니다.

## Google Sheets API Key 발급 방법

### 1. Google Cloud Console 프로젝트 생성

1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 상단 프로젝트 선택 드롭다운 → **새 프로젝트** 클릭
3. 프로젝트 이름 입력 후 **만들기**

### 2. Google Sheets API 활성화

1. 좌측 메뉴 **API 및 서비스 → 라이브러리** 이동
2. 검색창에 `Google Sheets API` 입력 후 선택
3. **사용 설정** 클릭

### 3. API 키 발급

1. 좌측 메뉴 **API 및 서비스 → 사용자 인증 정보** 이동
2. 상단 **+ 사용자 인증 정보 만들기 → API 키** 클릭
3. 생성된 키(`AIza...` 형식)를 복사

### 4. API 키 제한 설정 (권장)

발급된 키 클릭 → **API 제한** 탭에서 `Google Sheets API`만 허용하도록 설정하면 보안이 강화됩니다.

### 5. 스프레드시트 공개 설정

플러그인은 API 키 인증만 사용하므로 대상 시트가 **링크가 있는 모든 사용자 — 뷰어** 이상으로 공유되어 있어야 합니다.

> 📎 공식 문서: [Google Sheets API 시작하기](https://developers.google.com/sheets/api/guides/concepts?hl=ko)
